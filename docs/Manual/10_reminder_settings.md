# 5.1 Reminder Settings

## Overview
**Function:** Reminder Settings allow you to configure the automated email engine that follows up with parents who have paid but haven't submitted their required forms yet.

## Configuring Automated Chasers
**Action:**
1. Navigate to **Settings > Reminder Settings** on the left menu.
2. Select the specific Camp you want to configure (e.g., "2026 Season").
3. Toggle "Enable Automated Reminders" to **ON**.
4. Set the **Reminder Cadence:** How many days should the system wait between sending emails? (e.g., Every 7 days).
5. Set the **Maximum Reminders:** What is the limit? (e.g., Stop sending after 3 attempts).
6. Click **Save Settings**.

**Impact:** Massively reduces administrative workload. Instead of manually clicking "Chase" on 50 different registrations every week, the system automatically runs in the background, consistently prodding parents to submit critical medical and dietary information.
