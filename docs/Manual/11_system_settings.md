# 5.2 System Settings

## Overview
**Function:** System Settings control the core technical configuration and administrative access for the entire Camp Command application.

## Core Configuration
**Action:**
1. Navigate to **Settings > System Config** on the left menu.
2. **Access Control:** Update the primary Admin Email Address and Support Email Address (used for replying to parents).
3. **App URL:** Set the exact web address where parents go to fill out their forms (e.g., `https://forms.swedishcamp.com`).
4. **Email Integration:** Enter your Brevo API key to enable outgoing emails.
5. Click **Save Changes**.

## Testing Email Delivery
**Action:**
1. In the System Config page, scroll down to the **Test Email Configuration** section.
2. Enter a test email address that you have access to.
3. Click **Send Test Email**.
4. Check your inbox to confirm the system's email engine is working smoothly.

**Impact:** This is the technical lifeline of the application. Proper configuration here ensures that the app is secure, automated emails actually reach parents' inboxes, and the links in those emails point to the correct live website.
